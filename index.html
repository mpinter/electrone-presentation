Node.js Foreign Function Interface
A set of methods to call a function from dynamic library (.dll in case of Windows) via javascript
You only build the node-ffi lib itself and provide the lib in compiled state for current architecture

FFI Disadvantages

no cross-platform build process (or you'll need to roll one on your own)
call overhead
debugging is horrible
